CMake 完整教程

本文档为开箱即用的 CMake 全流程参考手册，所有章节均配套可直接运行的示例与操作步骤，配置项目时直接对照查阅即可。

一、基础认知与安装

1.1 什么是 CMake

CMake 是跨平台的构建系统生成工具，它不直接编译源代码，而是根据 CMakeLists.txt 配置文件，生成对应平台的原生构建脚本：

  • Linux/macOS：生成 Makefile
  • Windows：生成 Visual Studio 工程 / MinGW Makefile
  • macOS：也可生成 Xcode 项目

最终由原生构建工具（make、msbuild 等）完成代码的编译与链接。

1.2 核心优势

  • 一套配置适配全平台，无需为每个系统单独写构建脚本
  • 语法比原生 Makefile 更简洁易维护，支持大型工程模块化管理
  • 内置海量第三方库查找模块，自动处理依赖路径
  • 主流 C/C++ 开源项目的标准构建方案，生态完善

1.4 核心概念

概念 说明
CMakeLists.txt CMake 的配置文件，项目的所有构建规则都写在这里，文件名大小写严格
源码外构建（Out-of-source build） 在独立的 build 目录中执行编译，中间文件与源码分离，强烈推荐
构建类型 Debug（带调试符号、无优化）、Release（最高优化、无调试信息）等
目标（Target） 可执行文件、静态库、动态库都称为目标，所有配置都围绕目标展开

4.6 关键字说明

  • PUBLIC：属性对自身和依赖它的目标都生效
  • PRIVATE：属性只对自身生效
  • INTERFACE：属性只对依赖它的目标生效

最佳实践：优先使用 target_include_directories / target_compile_options 等目标级命令，比全局的 include_directories 更安全，避免不同模块互相干扰。

8.6 常见报错排查思路

  1. 找不到源文件：检查路径拼写、是否为相对路径（相对当前 CMakeLists.txt 所在目录）
  2. 找不到头文件：确认 include_directories 或 target_include_directories 路径正确
  3. 链接错误 undefined reference：确认库已链接、库架构匹配、符号存在
  4. find_package 失败：手动指定 XXX_DIR 为库的配置文件目录，或安装对应开发包
  5. 疑难杂症：删除 build 目录，重新执行 cmake ..，排除缓存干扰

九、完整命令语法速查

9.1 项目基础

命令 语法 作用
cmake_minimum_required cmake_minimum_required(VERSION x.x) 指定最低 CMake 版本
project project(名称 语言) 定义项目名与支持语言（C/CXX/ 不填 = 全支持）
set set(变量名 值) 定义 / 修改变量

9.2 目标定义

命令 语法 作用
add_executable add_executable(目标名 源文件...) 生成可执行文件目标
add_library add_library(库名 类型 源文件...) 生成库目标，类型：STATIC静态 /SHARED动态 /OBJECT对象库
add_subdirectory add_subdirectory(子目录路径) 引入子目录，执行其中的 CMakeLists.txt

9.3 头文件与路径

命令 语法 作用
include_directories include_directories(路径...) 全局添加头文件搜索路径
target_include_directories target_include_directories(目标 作用域 路径) 为指定目标添加头文件路径
link_directories link_directories(路径...) 全局添加库文件搜索路径

9.4 链接与编译属性

命令 语法 作用
target_link_libraries target_link_libraries(目标 作用域 库...) 为目标链接依赖库
target_compile_options target_compile_options(目标 作用域 选项...) 为目标添加编译选项
target_compile_definitions target_compile_definitions(目标 作用域 宏...) 为目标添加宏定义

9.5 流程控制

命令 语法 作用
if/elseif/else/endif if(条件) ... endif() 条件分支
option option(名称 描述 默认值) 定义布尔开关选项
foreach/endforeach foreach(变量 列表) ... endforeach() 循环遍历

9.6 依赖查找

命令 语法 作用
find_package find_package(包名 模式) 查找第三方依赖库
find_library find_library(变量 库名 路径) 查找指定库文件
find_path find_path(变量 头文件名 路径) 查找指定头文件所在路径

9.7 调试与输出

命令 语法 作用
message message(级别 内容) 输出日志信息
file file(操作 目标 内容) 文件读写、扫描、复制等操作
