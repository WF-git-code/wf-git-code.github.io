八、常见问题解决

  1. 提交信息写错了：git commit --amend
  2. 误删文件想恢复：git checkout HEAD 文件名
  3. 合并冲突：编辑冲突文件解决冲突后，执行git add 冲突文件和git commit
  4. 想撤销已经推送到远程的提交：先本地回滚，再强制推送git push -f origin 分支名（谨慎使用，会覆盖远程历史）
